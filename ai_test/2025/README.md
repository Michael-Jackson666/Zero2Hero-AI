## 🎯 使用说明

### 查看题目
每个日期文件夹包含当天的机试题目及解答：
- `.py` 文件：Python代码实现
- `.txt` 文件：测试用例或输入数据
- `.md` 文件：题目说明或算法解释

### 运行代码
```bash
# 进入具体日期目录
cd 10/17/

# 运行Python代码
python alarm_detail.py < test_input.txt

# 或使用测试文件
python backpropagation.py < test_backprop.txt
```

### 学习建议
1. 先阅读题目要求和测试用例
2. 尝试自己实现解决方案
3. 对比参考代码，理解优化思路
4. 关注代码中的**求解思路**部分
